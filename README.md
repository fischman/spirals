# Natural Spirals

Pure HTML/JS/CSS interactive visualization of naturally occurring 2D spirals,
with points plotted at integer arc-length distances from the origin.

Open `index.html` in a browser — no build step, no dependencies.

## Spirals

| Type | Formula | Seen in |
|------|---------|--------|
| Golden | r = a·φ^(2θ/π) | Nautilus shells, galaxies |
| Archimedean | r = aθ | Watch springs, coiled rope |
| Fermat's | r = a√θ | Sunflower seed heads |
| Hyperbolic | r = a/θ | Moth flight paths |
| Lituus | r = a/√θ | Magnetic field lines |

## Point subsets

Highlight integer arc-length points by:
all, odd, even, perfect squares, primes, Fibonacci, triangular numbers.
